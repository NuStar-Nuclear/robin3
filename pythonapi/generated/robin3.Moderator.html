robin3.Moderator¶

class robin3.Moderator(name, temperature=300.0, boron=0.0, pressure=0.101325)¶

Water used as moderator material.

This class is used to create water moderator materials by ROBIN3’s Python API, and to inspect attributes such as nuclide number densities, macro cross sections.

Parameters

  • name (str) – A unique string name to identify the material.

  • temperature (float, optional) – Temperature of the moderator material in Kelvin. (300 by default).

  • boron (float, optional) – Boron concentration of the moderator material in ppm. (0.0 by default).

  • pressure (float, optional) – Pressure of the moderator material in MPa. (0.101325 by default).
