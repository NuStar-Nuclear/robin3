robin3.Plotter¶

class robin3.Plotter(input_model, generate_geometry_data=True, assembly=None, result_data=None, num_process=32, write_json_for_debug=True, track=False, cyclic_track=False, quadrature='ALL', polar=0, interval=1)¶

A plotter used to generate visualization data.

The geometry and material data of the model should be defined either in a dict or a input(output) file. ROBIN3 performs geometrical computation and track generation which can be time-consuming before plotting, and caches the processed data for plotting.

Parameters

  • model (dict or str) – A dict which defined the model or a file name with path in which the model defined, both input yaml (json) file or output json file are acceptable.

  • assembly (str, optional) – The name of target assembly which you want to plot, only used when you want to plot a single assembly defined in a whole core model input file (None by default).

  • num_process (int, optional) – Number of process used to perform geometry computation. Note that geometry computations are parallelized in assembly level by multi-processing, which means this argument only works for core model ploting (number_of_processors_availabe by default).

  • track (bool, optional) – Whether to generate tracks to plot (False by default).

  • cyclic_track (bool, optional) – Whether to generate cyclic tracks to plot (False by default).

  • quadrature (str, optional) – Which quadrature used to generate tracks, a choice of [‘ALL’, ‘FAST’, ‘THERMAL’, ‘EPITHERMAL’] (‘ALL’ by default).

  • polar (int, optional) – Which polar level used to generate tracks (0 by default).

  • interval (int, optional) – The azimuth interval used to generate tracks (1 by default).

Examples

>>> import robin3
>>> import yaml
>>> with open('mini_core.yaml') as f:
>>>     model = yaml.load(f.read(), Loader=yaml.Loader)
>>> plotter = robin3.Plotter(model)
>>> plotter = robin3.Plotter('mini_core.yaml', assembly='B_00', track=True)

Plots a 2D heatmap of assembly power distribution in the geometry.

  • save_animation_to (str, optional) – Name of file used to save the burnup animation, note only ‘.gif’ or ‘.mp4’ suffix supported (‘burnup_animation.gif’ by default).

  • show_plot (bool, optional) – Whether to show the plot immediately (True by default).

  • save_fig (bool, optional) – Whether to save the current figure (False by default).

  • render_to_html (bool, optional) – Whether to render the plot to a html file for interactive visualization (False by default).

  • show_plot (bool, optional) – Whether to show the plot immediately (True by default).

  • save_fig (bool, optional) – Whether to save the current figure (False by default).
