robin3.ResultProcessor¶

class robin3.ResultProcessor(result_file)¶

A result processor used to collecting result data and doing some processing.

Parameters

result_file (str) – The HDF5 format result file with path produced by robin3 simulation.

result_file¶

The HDF5 format result file with path produced by robin3 simulation.

Type

str

result_data¶

A python list which hold all the result data.

Type

list

input_model¶

A python dict containing the input model.

Type

dict

is_macro_xs_case¶

Whether is macro-xs case.

Type

bool

is_case_matrix¶

Whether is single assembly case matrix case.

Type

bool

is_core_case¶

Whether is core simulation case.

Type

bool

generate_plotter(result_data=None, generate_geometry_data=True, num_process=32)¶

Generate a robin3.Plotter instance used to visualize the result data.

Parameters

  • result_data (list, optional) – A list contain all the result data that we want to visualize.

  • num_process (int, optional) – Number of process used to perform geometry computation. Note that geometry computations are parallelized in assembly level by multi-processing, which means this argument only works for core model ploting (number_of_processors_availabe by default).

Returns

Return type

robin3.Plotter

Examples

>>> import robin3
>>> result = robin3.ResultProcessor('mini_core.h5')
>>> plotter = result.generate_plotter(num_process=4)
>>> plotter.plot_geometry()
