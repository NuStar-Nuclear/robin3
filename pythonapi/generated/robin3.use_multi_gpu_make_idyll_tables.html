robin3.use_multi_gpu_make_idyll_tables¶

robin3.use_multi_gpu_make_idyll_tables(input_dir, output_dir=None, idyll_addtional_data=None, from_robin2_inp=False, library=None, verbose=False)¶

Use multi-GPU and multi-processing to make idyll tables.

Parameters

  • input_dir (str) – The directory contains the input files, both robin2’s .inp and robin3’s yaml or json format are acceptable.

  • output_dir (str, optinal) – The directory used to output table file, input directory will be used if None (None by default).

  • idyll_addtional_data (dict, optional) – A dict with data which will be added to the attributes of the idyll table file, nothing will be added if None (None by default).

  • from_robin2_inp (bool, optional) – Whether making table from robin2’s .inp format input file, robin3 will translate robin2’s .inp format file to .yaml input file before making idyll tables if True (False by default).

  • library (str, optional) – The nuclear data library file name with path, default nuclear data library file installed in package will be used if None (None by default).

  • verbose (bool, optional) – Whether to print iteration info to the logger, a progress bar will be used if False (False by default).

Returns

Return type

None

Examples

>>> import robin3
>>> robin3.use_multi_gpu_make_idyll_tables('./')
