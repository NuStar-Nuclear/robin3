robin3.Material¶

class robin3.Material(**kwargs)¶

A material composed by elements.

This Material class is used to create materials by ROBIN3’s Python API, and to inspect attributes such as nuclide number densities, macro cross sections.

Parameters

  • name (str) – A unique string name to identify the material.

  • input_type (str) – A choice of [“BY_WEIGHT_PERCENT”, “BY_VOLUME_PERCENT”, “BY_NUMBER_DENSITY”], by which way to define the material.

  • elements (list of str) –

    A list of string names of elements which compose this material, element here is a material as well, element material should be predefined either by user or contained in get_predefined_materials. Note:

    1. Enriched “UO2” can be defined use “UO2_enrichment” conveniently. For example, “UO2_3.2” means material UO2 with 3.2 enrichment.

    2. If input_type is “BY_VOLUME_PERCENT”, elements here should be materials that you have defined by weight percents.

  • number_densities (list of float, optional) – A list of float representing the nuclide number densities of each element, units in /barn/cm, required when input_type is “BY_NUMBER_DENSITY” (None by default).

  • weight_percents (list of float, optional) – A list of float representing the weight percents of each element, required when input_type is “BY_WEIGHT_PERCENT”. Note sum of weight_percents should be range in 0.999~1.0, the rest that less than 1.0 will be treated as impurity (None by default).

  • volume_percents (list of float, optional) – A list of float representing the volume percents of each element, required when input_type is “BY_VOLUME_PERCENT”. Note sum of volume_percents should be equal to 1.0 (None by default).

  • temperature (float, optional) – Temperature of the material in Kelvin. Default temperature 300K will be used if None (None by default).

  • density (float, optional) – A float, density of this material, units in g/cm^3, required when input_type is “BY_WEIGHT_PERCENT” (None by default)
