%matplotlib inline

Defining Materials¶

First we define 3 kinds of UO2 materials with different enrichments：

[2]:

materials = [
    {
        "name": "fuel_uo2_{}".format(enrichment),
        "input_type": "BY_WEIGHT_PERCENT",
        "elements": ["UO2_{}".format(enrichment)],
        "weight_percents": [1.0],
        "density": 10.3
    } for enrichment in [1.9, 2.5, 3.2]]

Then we define 2 kinds of Gd2O3-UO2 materials with different enrichments:

[3]:

materials.extend([{

Defining Pin Structures¶

First we define guide tube:

[4]:

guide_tube = {
    "name": "GT",
    "primitives": [
        {"type": "CIRCLE", "radius": 0.5615},
        {"type": "CIRCLE", "radius": 0.6120}
    ],
    "regions": [
        {"material": "MOD", "space": "1"},
        {"material": "ZIRLO", "space": "2-1"}
    ]
}

Then define different kinds of normal fuel pins：

[5]:

normal_fuel_pins = [{
    "name": name,
    "primitives": [
        {"type": "CIRCLE", "radius": 0.4096},
        {"type": "CIRCLE", "radius": 0.4180},
        {"type": "CIRCLE", "radius": 0.4750},
    ],
    "regions": [
        {"material": fuel, "space": "1"},
        {"material": "HE", "space": "2-1"},
        {"material": "ZIRLO", "space": "3-2"},
    ]
} for name, fuel in [("F19", "fuel_uo2_1.9"),
                     ("F25", "fuel_uo2_2.5"),
                     ("F32", "fuel_uo2_3.2")]]

Then define different kinds of Gd-bearing fuel pins：

[6]:

gd_bearing_fuel_pins = [{
    "name": name,
    "meshing": {
        "type": "SECTORS",
        "num_azimuths": 8,
        "radii": [np.sqrt(0.4096**2/10*(n+1)) for n in range(9)]
    },
    "primitives": [
        {"type": "CIRCLE", "radius": 0.4096},
        {"type": "CIRCLE", "radius": 0.4180},
        {"type": "CIRCLE", "radius": 0.4750},
    ],
    "regions": [
        {"material": fuel, "space": "1"},
        {"material": "HE", "space": "2-1"},
        {"material": "ZIRLO", "space": "3-2"},
    ]
} for name, fuel in [("G07", "fuel_gd_0.714"),
                     ("G19", "fuel_gd_1.9")]]

Put all pins into a list:

[7]:

pins = [guide_tube]
pins.extend(normal_fuel_pins)
pins.extend(gd_bearing_fuel_pins)

Defining Core¶

Core is used to define core model, note that the null character string “” should be used as a placeholder in assembly_map definition.

[9]:

core = {
    "name": "YanLong",
    "type": "SQUARE",
    "symmetry": 8,
    "assembly_map": [
           "",   "",   "","A32","A32","A32",   "",   "",   "",
           "",   "","A32","A19","A25","A19","A32",   "",   "",
           "","A32","A19","A25","A19","A25","A19","A32",   "",
        "A32","A19","A25","A19","A25","A19","A25","A19","A32",
        "A32","A19","A25","A19","A25","A19","A25","A19","A32",
        "A32","A19","A25","A19","A25","A19","A25","A19","A32",
           "","A32","A19","A25","A19","A25","A19","A32",   "",
           "",   "","A32","A19","A25","A19","A32",   "",   "",
           "",   "",   "","A32","A32","A32",   "",   "",   "",
    ],
    "reflector": {
        "baffles": [
            {"thickness": 1.0, "material": "MOD"},
            {"thickness": 2.0, "material": "SS304", "tag": "baffles"}
        ],
        "primitives": [
            {"type": "CIRCLE", "radius": 115},
            {"type": "CIRCLE", "radius": 125}
        ],
        "regions": [
            {"material": "MOD", "space": '1'},
            {"material": "SS304", "space": '2-1', "tag": "bucket"}
        ]
    }
}

Note here we tags two material regions in the reflector, “baffle” and “bucket”, neutron fluxes data in the tagged regions are edited specifically.

Defining State Points¶

[10]:

state_points = {
    "boron_concentration": 0.0,
    "moderator_temperature": 348.0,
    "fuel_temperature": 500.0,
    "system_pressure": 1.0,
}

Note here we do not define power_density and burnup_points, which means that ROBIN3 will only perform single state point caculation.

Composing Input Model¶

[11]:

model = {
    "materials": materials,
    "structures": pins,
    "assemblies": assemblies,
    "core": core,
    "state_points": state_points,
}

Model Visualization¶

displaying different kinds of assemblies with fine meshes one by one by using robin3.Plotter:

[23]:

for assembly in ["A19", "A25", "A32"]:
    plotter = robin3.Plotter(model, assembly=assembly)
    fig = plotter.plot_geometry(show_meshes=True)

We can display the 1/8 core model:

[24]:

plotter = robin3.Plotter(model)
fig = plotter.plot_geometry()

Plotting the 1/8 core with MOC meshes, ROBIN3 will automatically quadruple the number of meshes of cells which across material boundary in reflector to improve the caculation accuracy.

[25]:

fig = plotter.plot_geometry(show_meshes=True)

Since we tagged baffle and bucket regions, we can plot them as well.

[26]:

fig = plotter.plot_tagged_regions()

Run ROBIN3¶

Run the simulation by using robin3.run() function：

[12]:

result = robin3.run(model, use_gpu=True)

display the summary of the simulation result：

[13]:

result.summary()

[13]:
burnup tmo tfu bor keff peak_pin_power std_pin_power

Result Visualization¶

We can plot the pin power distribution：

[16]:

plotter = result.generate_plotter()
fig = plotter.plot_pin_power()

Plotting the pin flux distribution of energy group 1 (fast group) and 2 (thermal group).

[17]:

for g in [1, 2]:
    fig = plotter.plot_pin_flux(group=g)

If we want to inspect the distribution in fine mesh level, fine mesh fluxes are accessible as well.

[18]:

for g in [1, 2]:
    fig = plotter.plot_mesh_flux(group=g)

We can get the neutron fluxes with energy in the tagged regions and plot them.

[13]:

result.neutron_fluxes_in_tagged_regions()
