Modeling VVER 1/6 Core¶

This notebook is intended to demonstrate how to use ROBIN3’s Python API to simulate a 2-D VVER 1/6 core.

[1]:

%matplotlib inline
from pprint import pprint
from copy import deepcopy

import matplotlib.pyplot as plt
plt.rcParams['figure.figsize'] = [20, 10]
robin3.config.USE_TQDM_NOTEBOOK = True

Defining Materials¶

Use two functions to define different enrichments UO2 fuel materials and Gd-bearing fuel materials.

[2]:

def uo2_fuel(enrichment):
    return {
        "name": "UO2_{}".format(enrichment),
        "elements": ["UO2_{}".format(enrichment)],
        "weight_percents": [1.0],
        "density": 10.27
    }

def gd_bearing_fuel(enrichment):
    return {
        "name": "UO2_{}_Gd".format(enrichment),
        "elements": ["UO2_{}".format(enrichment), "GD2O3"],
        "weight_percents": [0.95, 0.05],
        "density": 9.4
    }

uo2_fuel_enrichments = [1.3, 2.2, 3.0, 3.6, 4.0, 4.4]
gd_bearing_fuel_enrichments = [2.4, 3.3, 3.6]
uo2_fuels = [uo2_fuel(enrichment) for enrichment in uo2_fuel_enrichments]
gd_bearing_fuels = [gd_bearing_fuel(enrichment) for enrichment in gd_bearing_fuel_enrichments]

Defining a materials list, and put fuel materials defined above into this list, then append 2 alloy materials.

Defining Pin Structures¶

Here we define pin structures.

def fuel_pin(enrichment):
     return {"name": "F{}".format(int(enrichment*10)),
            "primitives": [
                {"type": "CIRCLE", "radius": 0.3785},
                {"type": "CIRCLE", "radius": 0.3865},
                {"type": "CIRCLE", "radius": 0.4550},
            ],
            "regions": [
                {"material": "UO2_{}".format(enrichment), "space": "1"},
                {"material": "HE", "space": "2-1"},
                {"material": "ZIRLO", "space": "3-2"},
            ]}

def gd_bearing_pin(enrichment):
    pin = fuel_pin(enrichment)
    pin["name"] = "B{}".format(int(enrichment*10))
    pin["regions"][0]["material"] = "UO2_{}_Gd".format(enrichment)
    pin["meshing"] = {"type": "SECTORS",
                     "num_azimuths": 6,
                     "radii": [np.sqrt(0.3785**2 / 10 * (n+1)) for n in range(9)]}
    return pin

Run ROBIN3¶

Running ROBIN3 from Python can be done using the robin3.run() function. This function allows you to use GPU by setting use_GPU=True if it needed, GPU will significantly reduce the running time.

[10]:

result = robin3.run(model, use_gpu=True)
print(result)

<robin3.result_process.ResultProcessor object at 0x7f63a67ebbe0>

The robin3.run() function return a robin3.ResultProcessor instance which can be initialized from the result file as well. All the result data can be accessed by the result_data attribute, and the result data are saved by a HDF5 file named by result_file attribute, summary result data such as keff can be inspected by method summary which return a pandas DataFrame.

[2]:

result = robin3.ResultProcessor('VVER.h5')
df = result.summary()
print(df)

Result Visualization¶

Result visualization from Python can be done using a robin3.Plotter instance generate by ResultProcessor’s method generate_plotter.

[3]:

plotter = result.generate_plotter()

From the output file, many result items can be displayed by the robin3.Plotter Class. Here we display the pin power distribution.

[8]:

fig = plotter.plot_pin_power()

We can plot the pin flux distribution of energy group 1 (fast group) and 2 (thermal group).

[12]:

for g in [1, 2]:
    fig = plotter.plot_pin_flux(group=g)

If we want to inspect the distribution in fine mesh level, fine mesh fluxes are accessible as well.

[9]:

for g in [1, 2]:
    fig = plotter.plot_mesh_flux(group=g)
